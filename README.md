# mf-to-spreadsheet

MoneyForwardのデータをSpreadSheetに連携するスクリプト。

## バッチの実行

```sh
docker compose up
```

## コンテナへの接続

```sh
docker compose exec app bash
```

## コンテナの停止

```sh
docker compose down
```
